## Amiga-Specific Features

This curriculum highlights the Amiga's unique characteristics:

- **Structured AMOS** - No line numbers, procedures, modern syntax
- **BOB system** - Hardware-accelerated Blitter OBjects for sprites
- **Paula audio** - 4-channel sample-based sound with volume/pitch control
- **Copper co-processor** - Display list programming for raster effects
- **Blitter** - Hardware block copying and fill operations for fast graphics
- **Double buffering** - Built-in flicker-free animation
- **Custom chipsets** - Direct access to Agnus, Denise, Paula
