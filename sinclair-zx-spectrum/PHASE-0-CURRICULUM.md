## Spectrum-Specific Features

This curriculum highlights the ZX Spectrum's unique characteristics:

- **Fast BASIC execution** - Programs run noticeably quicker than on C64
- **Direct graphics commands** - PLOT, DRAW, CIRCLE work immediately
- **PRINT AT positioning** - No memory address calculations needed
- **Keyword entry system** - Press P for PRINT, faster than typing
- **Attribute system** - Colour clash teaches creative problem-solving
- **BEEP sound** - Simple but effective audio programming
