## Technical Summary

### Key ZX Spectrum Features
- Platformer + shooter hybrid
- 8-way shooting system
- Multiple weapon types
- Boss battles
- Two-player co-op

### Code Pattern
```z80
; 8-way aim direction
get_aim_direction:
    ld a,(keys_held)
    ld b,0              ; Default: right
    bit KEY_UP,a
    jr z,.not_up
    ld b,DIR_UP
.not_up:
    bit KEY_DOWN,a
    jr z,.not_down
    ld b,DIR_DOWN
.not_down:
    bit KEY_LEFT,a
    jr z,.not_left
    ld a,b
    add DIR_LEFT        ; Combine with vertical
    ld b,a
.not_left:
    ; Similar for right
    ld a,b
    ld (aim_dir),a
    ret
```
