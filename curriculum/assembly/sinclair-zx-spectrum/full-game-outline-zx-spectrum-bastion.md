# Game 15: Bastion - Complete 512-Unit Outline

## Game Overview
**Platform:** Sinclair ZX Spectrum
**Genre:** Isometric exploration
**Concept:** Knight Lore-style isometric adventure - the prestige Spectrum technique. Explore a vast castle, solve puzzles, collect items, transform between forms, and break the curse before time runs out.

**Technical Focus:**
- Isometric projection mathematics
- Depth sorting (painter's algorithm)
- Object interaction system
- Transformation mechanics
- Time-based gameplay
- Large connected world (100+ rooms)
- 128K enhanced version

## Technical Summary

### Key ZX Spectrum Features
- Isometric projection mathematics
- Depth sorting (painter's algorithm)
- Transformation mechanics
- 100+ room world
- 128K AY sound support
- Time-based gameplay

### Code Pattern
```z80
; Isometric world-to-screen conversion
iso_to_screen:
    ; screen_x = (world_x - world_y) * 2 + SCREEN_CENTER_X
    ld a,(world_x)
    ld b,a
    ld a,(world_y)
    sub b               ; world_x - world_y
    add a,a             ; * 2
    add a,SCREEN_CENTER_X
    ld (screen_x),a
    ; screen_y = (world_x + world_y) - world_z + SCREEN_CENTER_Y
    ld a,(world_x)
    ld b,a
    ld a,(world_y)
    add a,b             ; world_x + world_y
    ld b,a
    ld a,(world_z)
    neg
    add a,b             ; - world_z
    add a,SCREEN_CENTER_Y
    ld (screen_y),a
    ret
```
