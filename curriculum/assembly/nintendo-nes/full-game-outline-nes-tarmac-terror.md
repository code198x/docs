## Technical Summary

### Key NES Features
- Pseudo-3D road rendering
- Sprite scaling simulation
- Road curves and hills
- Traffic system
- Split-screen multiplayer

### Code Pattern
```asm
; Road width calculation per scanline
calc_road_width:
    lda scanline
    sec
    sbc #HORIZON         ; Distance from horizon
    beq .at_horizon
    ; width = BASE_WIDTH * 256 / distance
    sta temp
    lda #0
    ldx #BASE_WIDTH
.div_loop:
    sec
    sbc temp
    bcc .div_done
    inx
    bne .div_loop
.div_done:
    stx road_width
.at_horizon:
    rts
```
