## Technical Summary

### Key NES Features
- Platformer + shooter hybrid
- 8-way aiming system
- Multiple weapon types
- Two-player cooperative
- Heavy sprite management

### Code Pattern
```asm
; 8-way bullet direction
fire_bullet:
    lda aim_direction
    asl a               ; x2 for table offset
    tax
    lda bullet_dx_table,x
    sta bullet_dx
    lda bullet_dy_table,x
    sta bullet_dy
    jsr spawn_bullet
    rts

bullet_dx_table:
    .db 3, 2, 0, -2, -3, -2, 0, 2  ; R, DR, D, DL, L, UL, U, UR
bullet_dy_table:
    .db 0, 2, 3, 2, 0, -2, -3, -2
```
