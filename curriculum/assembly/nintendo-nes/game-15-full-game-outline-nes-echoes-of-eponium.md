## Technical Summary

### Key NES Features
- MMC3 mapper mastery
- Scanline IRQ effects
- Large connected world
- Ability-gating design
- Battery-backed saves
- Multiple endings

### Code Pattern
```asm
; MMC3 IRQ scanline split
setup_irq:
    lda #STATUS_BAR_HEIGHT-1
    sta MMC3_IRQ_LATCH      ; Set scanline counter
    sta MMC3_IRQ_RELOAD     ; Reload counter
    lda MMC3_IRQ_ENABLE     ; Enable IRQ
    rts

irq_handler:
    pha
    sta MMC3_IRQ_DISABLE    ; Acknowledge IRQ
    ; Switch to game scroll
    lda camera_x
    sta PPUSCROLL
    lda camera_y
    sta PPUSCROLL
    ; Re-enable for next frame
    sta MMC3_IRQ_ENABLE
    pla
    rti
```
