# Game 10: Wanderer - Complete 256-Unit Outline

## Game Overview
**Platform:** Commodore Amiga
**Genre:** Scrolling platformer
**Concept:** Epic scrolling platformer with vast worlds to explore. Building on Parallax's scrolling expertise, create a full platformer with multiple themed worlds, bosses, and commercial-quality polish.

**Technical Focus:**
- Horizontal scrolling platformer
- Camera systems with dead zones
- Large level compression
- Parallax backgrounds
- Moving platforms
- Enemy AI in scrolling world

## Technical Summary

### Key Amiga Features
- Hardware scrolling
- Blitter tile rendering
- Multi-layer parallax
- Copper colour effects
- BOB sprites
- Paula MOD playback

### Code Pattern
```asm
; Camera dead zone following
update_camera:
    move.w  player_x,d0
    sub.w   camera_x,d0
    cmp.w   #DEAD_LEFT,d0
    blt     scroll_left
    cmp.w   #DEAD_RIGHT,d0
    bgt     scroll_right
    rts
```
