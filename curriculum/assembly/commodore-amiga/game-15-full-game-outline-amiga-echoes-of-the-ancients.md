# Game 15: Echoes of the Ancients - Complete 512-Unit Outline

## Game Overview
**Platform:** Commodore Amiga
**Genre:** Metroidvania
**Concept:** Connected world exploration with ability-gating on Amiga hardware. Multi-layer parallax, Blitter-powered gameplay, and Copper effects create an atmospheric adventure. Battery-backed save system and multiple endings.

**Technical Focus:**
- Large connected world map
- Non-linear progression
- Ability-gating design
- Multi-layer parallax per region
- Advanced Blitter techniques
- Copper effects per zone
- Disk-based save system
- AGA enhanced version

## Technical Summary

### Key Amiga Features
- Large connected world
- Multi-layer parallax per region
- Copper effects per zone
- Blitter-powered gameplay
- Disk-based save system
- AGA enhanced version

### Code Pattern
```asm
; Region-specific Copper palette
setup_region_copper:
    lea     copper_list,a0
    move.w  region,d0
    mulu    #COPPER_SIZE,d0
    lea     region_coppers(pc,d0.l),a1
    ; Copy region-specific colour entries
    move.w  #PALETTE_SIZE-1,d7
.copy_loop:
    move.l  (a1)+,(a0)+     ; Copper WAIT/MOVE pairs
    dbf     d7,.copy_loop
    ; Add gradient entries
    jsr     build_gradient_copper
    ; Terminate list
    move.l  #$fffffffe,(a0)
    rts
```
