# Game 14: Fortress - Complete 256-Unit Outline

## Game Overview
**Platform:** Commodore Amiga
**Genre:** Isometric action-adventure
**Concept:** Knight Lore/Cadaver-style isometric adventure with exploration, puzzles, combat, and Amiga-quality audiovisuals. Master isometric projection and depth sorting with the Blitter.

**Technical Focus:**
- Isometric projection mathematics
- Depth sorting (painter's algorithm)
- Blitter-based isometric tile rendering
- Z-height collision detection
- Large connected world
- Copper colour effects per zone

## Technical Summary

### Key Amiga Features
- Isometric projection mathematics
- Blitter-based depth sorting
- Z-height collision detection
- Copper colour effects per zone
- Large connected world

### Code Pattern
```asm
; Isometric world-to-screen conversion
iso_to_screen:
    ; screen_x = (world_x - world_y) * TILE_WIDTH/2
    move.w  world_x,d0
    sub.w   world_y,d0
    muls    #TILE_WIDTH/2,d0
    move.w  d0,screen_x
    ; screen_y = (world_x + world_y) * TILE_HEIGHT/2 - world_z
    move.w  world_x,d0
    add.w   world_y,d0
    muls    #TILE_HEIGHT/2,d0
    sub.w   world_z,d0
    move.w  d0,screen_y
    rts
```
