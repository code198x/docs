# Game 12: Highway - Complete 256-Unit Outline

## Game Overview
**Platform:** Commodore Amiga
**Genre:** Pseudo-3D racing
**Concept:** Outrun-style racing with Copper-based road rendering, sprite scaling, and high-speed action. Multiple tracks, traffic, and split-screen two-player mode.

**Technical Focus:**
- Perspective projection mathematics
- Copper-based road rendering
- Sprite scaling simulation
- Road curves and hills
- Traffic system
- Split-screen multiplayer

## Technical Summary

### Key Amiga Features
- Copper-based road rendering
- Perspective mathematics
- Pre-scaled sprite system
- Split-screen multiplayer
- Dynamic time-of-day

### Code Pattern
```asm
; Road curve offset per scanline
calculate_road_offset:
    move.w  curve_value,d0      ; Current curve
    move.w  scanline,d1
    sub.w   #HORIZON,d1         ; Distance from horizon
    muls    d0,d1               ; Scale by distance
    asr.w   #8,d1               ; Normalise
    add.w   road_base_x,d1      ; Add base position
    move.w  d1,(a0)+            ; Store for Copper
    rts
```
