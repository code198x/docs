## Technical Summary

### Key C64 Features at Mastery
- Demo-scene effects (FLD, DYCP, AGSP, tech-tech)
- Advanced SID (filters, ring mod, multi-speed, digi)
- Direct 1541 programming with custom fast loader
- IRQ loader with music during load
- Memory optimisation for maximum content
- All VIC-II tricks combined
- Commercial-quality production

### Code Pattern
```asm
; FLD (Flexible Line Distance) effect
fld_effect:
    ; Modify $D011 mid-frame for screen warp
    lda #$1b        ; Standard screen config
    sta $d011

    ; Wait for specific raster line
.wait_line:
    lda $d012
    cmp #EFFECT_LINE
    bne .wait_line

    ; Apply FLD - skip badlines
    lda $d011
    and #$f8
    ora fld_offset
    sta $d011

    ; Continue rendering
    rts
```
