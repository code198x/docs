# Game Comparisons: Curriculum to Commercial

**Purpose:** Map each curriculum game to comparable commercial titles, showing learners what they're building towards and the historical context of each genre.

**Last Updated:** 2026-01-18

---

## How to Use This Document

Each curriculum game entry includes:
- **You're Building:** The curriculum game
- **Inspired By:** Direct historical inspirations
- **Genre Siblings:** Contemporary games in the same style
- **Modern Descendants:** Games that carry the torch today
- **Vault Entries Needed:** Games requiring Vault entries
