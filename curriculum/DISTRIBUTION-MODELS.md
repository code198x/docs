# Distribution Models: How Games Reached Players

**Purpose:** Document the various ways software was distributed - from budget games to shareware to public domain - and how these models shaped the industry.

**Last Updated:** 2026-01-18

---

## Why Distribution Models Matter

How games reached players affected:
1. **Who could afford games** - Price determined audience
2. **Developer economics** - Revenue models varied wildly
3. **Game quality** - Budget affected scope
4. **Discovery** - How players found new games
5. **Piracy relationship** - Price vs copying calculus

---

## Part 1: Full-Price Retail

### The Standard Model

| Aspect | Typical (UK, 1985) |
|--------|-------------------|
| **Price** | £7.95-£9.95 (tape), £12.95-£14.95 (disk) |
| **Margin** | ~30% to retailer, ~20% to distributor |
| **Developer** | ~20-30% royalty or work-for-hire |
| **Publisher** | Marketing, manufacturing, distribution |

### Retail Channels

| Channel | Examples |
|---------|----------|
| **High street** | WH Smith, Boots, John Menzies |
| **Specialists** | Computer shops, game shops |
| **Mail order** | Magazine advertisements |
| **Department stores** | Woolworths, Debenhams |

### Publisher Hierarchy (UK)

| Tier | Publishers |
|------|------------|
| **Major** | Ocean, US Gold, Elite, Mirrorsoft |
| **Mid-tier** | Gremlin, Hewson, Firebird |
| **Smaller** | Dozens of bedroom operations |

## Part 2: Budget Games

### The Budget Revolution

**Budget games** typically cost £1.99-£2.99 - transforming the market.

| Publisher | Price Point | Era |
|-----------|-------------|-----|
| **Mastertronic** | £1.99 | 1984-1991 |
| **Firebird Silver** | £2.50 | 1984-1988 |
| **Ricochet** | £2.99 | 1987+ |
| **Kixx** | £2.99 | 1988+ |
| **Hit Squad** | £2.99 | 1988+ |
| **Encore** | £2.99 | 1988+ |

### Mastertronic: The Pioneer

| Aspect | Notes |
|--------|-------|
| **Founded** | 1984, by Martin Alper |
| **Innovation** | £1.99 price point |
| **Strategy** | Volume over margin |
| **Distribution** | Newsagents, petrol stations |
| **Quality** | Variable but some gems |
| **Merged** | With Virgin (1988) |

**Key insight:** Games at newsagent price = impulse purchase.

### How Budget Economics Worked

| Factor | Full Price | Budget |
|--------|------------|--------|
| **Price** | £9.95 | £1.99 |
| **Units to profit** | 5,000 | 25,000 |
| **Development budget** | £5,000-£20,000 | £500-£2,000 |
| **Break-even** | Harder | Achievable |

### Budget Quality Spectrum

| Quality | Examples |
|---------|----------|
| **Gems** | The Last Ninja (re-release), various |
| **Solid** | Many playable games |
| **Shovelware** | Quick cash-ins |
| **Re-releases** | Full-price games at budget |

### Re-release Labels

Publishers created budget labels for back catalogue:
- **Kixx** (US Gold)
- **Hit Squad** (Ocean)
- **Encore** (Elite)
- **Ricochet** (Firebird → MicroProse)

## Part 3: Shareware

### The Shareware Model

| Principle | Implementation |
|-----------|----------------|
| **Try before buy** | First episode free |
| **Honor system** | "Please register" |
| **Distribution** | Copy freely, registration costs |
| **Tiers** | Free → Registered → Commercial |

### US Shareware Giants

#### Apogee Software

| Aspect | Notes |
|--------|-------|
| **Founded** | 1987, Scott Miller |
| **Model** | Episodic shareware |
| **Games** | Commander Keen, Duke Nukem, Wolfenstein 3D |
| **Innovation** | "Apogee model" - free episode 1 |

The Apogee model: Give away Episode 1, sell Episodes 2-3.

#### id Software

| Aspect | Notes |
|--------|-------|
| **Founded** | 1991 (from Softdisk) |
| **Shareware hits** | Wolfenstein 3D, Doom, Quake |
| **Innovation** | Engine licensing |
| **Relationship** | Published through Apogee initially |

**Doom** (1993) was distributed via shareware - millions of copies of Episode 1.

#### Epic MegaGames

| Aspect | Notes |
|--------|-------|
| **Founded** | 1991, Tim Sweeney |
| **First game** | ZZT (shareware) |
| **Later** | Unreal, Unreal Engine |
| **Model** | Shareware → commercial → engine licensing |

### Shareware Distribution

| Method | Notes |
|--------|-------|
| **BBS** | Download from bulletin boards |
| **Disk vendors** | Companies selling shareware disks |
| **Magazine coverdisks** | Shareware on cover media |
| **Friend copies** | "Copy this to a friend" |
| **CompuServe/AOL** | Online services |

### Registration Psychology

| Incentive | Example |
|-----------|---------|
| **More episodes** | Doom: 1 free, 2 paid |
| **Extra features** | Registered version extras |
| **No nag screens** | Remove "please register" |
| **Printed manual** | Physical goods |
| **Support** | Access to help |

## Part 4: Public Domain (PD)

### What PD Meant

**Public Domain** in gaming context meant:
- Free to copy
- Free to distribute
- No payment required
- Often amateur productions
- Sometimes abandoned commercial software

### PD Libraries

| Library | Platform | Notes |
|---------|----------|-------|
| **17-Bit Software** | Amiga | Huge catalogue |
| **AMOS PD** | Amiga | AMOS-made games |
| **PC-SIG** | PC | Large US library |
| **Various UK libraries** | Multi | Mail order |

### PD Distribution

| Method | Cost |
|--------|------|
| **Disk copying fee** | £1-2 per disk |
| **Magazine coverdisks** | Free with magazine |
| **BBS download** | Phone line costs |
| **User groups** | Membership |

### PD Quality

| Type | Quality |
|------|---------|
| **Utilities** | Often excellent |
| **Games** | Variable, some gems |
| **Demos** | Scene productions |
| **Source code** | Educational value |

### Notable PD Games

| Game | Platform | Notes |
|------|----------|-------|
| **Llamatron** | Amiga | Shareware/PD hybrid |
| **Galaga '92** | Amiga | Excellent clone |
| **Various** | All platforms | Too many to list |

## Part 5: Licenseware

### The Licenseware Model

**Licenseware** = Commercial PD - authors got royalties:

| Aspect | Notes |
|--------|-------|
| **Price** | £2-5 typically |
| **Distribution** | PD libraries sold them |
| **Royalties** | Author received payment |
| **Quality control** | Some curation |

### Major Licenseware Labels

| Label | Platform | Notes |
|-------|----------|-------|
| **AMOS PD Licenseware** | Amiga | AMOS games |
| **F1 Licenseware** | Amiga | Various |
| **Softville** | Amiga | Various |
| **Robtek** | Amiga | Various |

### Why Licenseware Mattered

| Benefit | For Whom |
|---------|----------|
| **Income** | Amateur developers |
| **Distribution** | Without publisher |
| **Low barrier** | Easy to release |
| **Quality middle ground** | Between PD and commercial |
