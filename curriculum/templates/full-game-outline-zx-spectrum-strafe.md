## Game Overview
**Platform:** Sinclair ZX Spectrum
**Genre:** Vertical scrolling shooter
**Concept:** 1942-style vertical shooter with plane formations, bullet patterns, weapon upgrades, and epic boss battles. 7+ stages of intense aerial combat with 128K AY sound support.

**Technical Focus:**
- Vertical software scrolling
- Formation enemy patterns
- Bullet pattern systems
- Multiple weapon types
- Multi-phase boss encounters
- 128K detection and AY sound

## Technical Summary

### Key ZX Spectrum Features
- Vertical software scrolling
- Formation enemy patterns
- Bullet pattern systems
- 128K AY sound support
- Bank switching basics

### Code Pattern
```z80
; Vertical scroll one row
scroll_up:
    ld hl,$4000+32      ; Source (row 1)
    ld de,$4000         ; Dest (row 0)
    ld bc,32*191        ; All but last row
    ldir                ; Fast block copy
    ; Clear bottom row
    ld hl,$4000+32*191
    ld de,$4000+32*191+1
    ld (hl),0
    ld bc,31
    ldir
    ret
```
