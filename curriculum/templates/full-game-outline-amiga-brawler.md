# Game 11: Brawler - Complete 256-Unit Outline

## Game Overview
**Platform:** Commodore Amiga
**Genre:** Side-scrolling beat 'em up
**Concept:** Streets of Rage / Final Fight style brawler with Y-depth sorting, combo system, multiple characters, and two-player co-op. Large animated sprites with melee combat.

**Technical Focus:**
- Y-axis depth sorting
- Large sprite animation
- Melee combat mechanics
- Combo systems
- Enemy coordination
- Two-player cooperative

## Technical Summary

### Key Amiga Features
- Y-depth sprite sorting
- Large animated sprites
- Blitter rendering
- Cooperative two-player
- MOD music

### Code Pattern
```asm
; Y-depth insertion sort
sort_sprites:
    lea     sprite_list,a0
    move.w  sprite_count,d7
    subq.w  #1,d7
.outer:
    move.w  d7,d6
    lea     sprite_list,a0
.inner:
    move.w  (a0),d0        ; Y of current
    move.w  4(a0),d1       ; Y of next
    cmp.w   d0,d1
    bge.s   .no_swap
    ; Swap entries
    movem.l (a0),d0-d1
    movem.l 4(a0),d2-d3
    movem.l d2-d3,(a0)
    movem.l d0-d1,4(a0)
.no_swap:
    addq.l  #4,a0
    dbf     d6,.inner
    dbf     d7,.outer
    rts
```
