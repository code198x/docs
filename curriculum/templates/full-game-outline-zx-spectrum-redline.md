# Game 11: Redline - Complete 256-Unit Outline

## Game Overview
**Platform:** Sinclair ZX Spectrum
**Genre:** Vertical scrolling racer
**Concept:** Championship racing with multiple tracks, AI opponents, weather effects, and damage system. Introduces tape loading fundamentals and TAP file creation.

**Technical Focus:**
- Vertical software scrolling
- Racing AI and overtaking
- Speed/physics simulation
- Track variety and curves
- Weather effects
- **Tape loading basics**
- **TAP file creation**

## Technical Summary

### Key ZX Spectrum Features
- Vertical software scrolling
- Racing physics
- AI opponent system
- Weather effects
- **Standard tape loading**
- **TAP file format**

### Code Pattern
```z80
; Standard tape save header
tap_header:
    db $00          ; Header block type
    db $03          ; CODE file type
    dm "SAVEGAME  " ; 10 char filename
    dw data_length  ; Data length
    dw load_address ; Load address
    dw $8000        ; Unused
```
