# Game 13: Onslaught - Complete 256-Unit Outline

## Game Overview
**Platform:** Commodore Amiga
**Genre:** Vertical scrolling shooter
**Concept:** 1942/SWIV-style vertical shooter with formation patterns, bullet patterns, ground and air targets, and epic multi-phase bosses. Intense action showcasing the Amiga's capabilities.

**Technical Focus:**
- Vertical hardware scrolling
- Formation enemy patterns
- Bullet pattern systems
- Ground and air target layers
- Multi-phase boss encounters
- Rank system (adaptive difficulty)

## Technical Summary

### Key Amiga Features
- Vertical hardware scrolling
- Formation enemy patterns
- Bullet pattern systems
- Ground/air target layers
- Multi-phase bosses
- Rank difficulty system

### Code Pattern
```asm
; Spiral bullet pattern
spawn_spiral_pattern:
    move.w  #16,d7              ; 16 bullets
    move.w  #0,d6               ; Starting angle
.loop:
    move.w  d6,d0
    jsr     sin_lookup          ; Get sine
    muls    #BULLET_SPEED,d0
    asr.l   #8,d0
    move.w  d0,bullet_dx(a0)
    move.w  d6,d0
    jsr     cos_lookup          ; Get cosine
    muls    #BULLET_SPEED,d0
    asr.l   #8,d0
    move.w  d0,bullet_dy(a0)
    add.w   #360/16,d6          ; Next angle
    jsr     next_bullet
    dbf     d7,.loop
    rts
```
