## Technical Summary

### Key NES Features
- Heavy sprite management
- Priority-based flicker
- Complex enemy AI
- Multiple game modes
- Full APU utilisation

### Code Pattern
```asm
; Sprite priority allocation
allocate_sprites:
    ; Player always first
    ldx #0
    jsr draw_player
    ; Sort enemies by distance
    jsr sort_enemies
    ; Draw until limit
@loop:
    cpx #MAX_SPRITES
    bcs @done
    jsr draw_next_enemy
    jmp @loop
@done:
    rts
```
