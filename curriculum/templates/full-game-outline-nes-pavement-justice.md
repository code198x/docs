## Game Overview
**Platform:** Nintendo Entertainment System
**Genre:** Side-scrolling beat 'em up
**Concept:** Double Dragon/River City Ransom-style brawler with Y-depth sorting, combo systems, weapon pickups, and two-player co-op. Vigilante action cleaning up the streets.

**Technical Focus:**
- Y-axis depth sorting
- Large animated sprites with rotation
- Melee combat hit detection
- Combo and juggle systems
- Multiple enemy AI coordination
- Two-player cooperative mode

## Technical Summary

### Key NES Features
- Y-axis depth sorting
- Large metasprite characters
- Melee combat system
- Sprite limit management
- Two-player cooperative

### Code Pattern
```asm
; Y-depth sprite sorting
sort_sprites:
    ; Bubble sort by Y position
    ldx #0
.outer:
    ldy #0
.inner:
    lda sprite_y,y
    cmp sprite_y+1,y
    bcc .no_swap
    ; Swap sprites
    jsr swap_sprite_data
.no_swap:
    iny
    cpy #SPRITE_COUNT-1
    bne .inner
    inx
    cpx #SPRITE_COUNT-1
    bne .outer
    rts
```
