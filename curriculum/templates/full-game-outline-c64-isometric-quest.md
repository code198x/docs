## Game Overview
**Platform:** Commodore 64
**Genre:** Isometric adventure
**Concept:** Knight Lore-style isometric adventure with exploration, puzzles, and combat. Master isometric projection mathematics, depth sorting, and the prestige technique that defined a generation of games.

**Technical Focus:**
- Isometric projection mathematics
- World-to-screen coordinate conversion
- Depth sorting (painter's algorithm)
- Isometric collision detection
- Z-height handling
- Large connected world

## Technical Summary

### Key C64 Features
- Isometric projection mathematics
- Depth sorting (painter's algorithm)
- Z-height collision detection
- Multi-zone world structure
- Save/password system

### Code Pattern
```asm
; Isometric world-to-screen conversion
iso_to_screen:
    ; screen_x = (world_x - world_y) * TILE_WIDTH/2
    lda world_x
    sec
    sbc world_y
    asl a               ; * TILE_WIDTH/2
    sta screen_x
    ; screen_y = (world_x + world_y) * TILE_HEIGHT/2 - world_z
    lda world_x
    clc
    adc world_y
    lsr a               ; * TILE_HEIGHT/2
    sec
    sbc world_z         ; Subtract height
    sta screen_y
    rts
```
