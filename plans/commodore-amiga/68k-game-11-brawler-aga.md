# Brawler AGA: 32-Unit Outline

**Game:** 11 AGA Enhancement
**Platform:** Commodore Amiga (A1200/A4000)
**Language:** 68000 Assembly
**Units:** 32 (2 phases)
**Prerequisite:** Brawler (OCS) complete
**Concept:** AGA-enhanced beat 'em up with larger, more colourful characters.

---

## Overview

Brawler AGA showcases what AGA means for character-focused games. Larger sprites with more colours, smoother animation, and richer stage backgrounds - the difference is immediately visible.

---

## AGA Enhancements

| Feature | OCS Version | AGA Version |
|---------|-------------|-------------|
| Character colours | 16 | 64+ |
| Background | 32 colours | 256 colours |
| Sprite detail | Limited shading | Full shading |
| Effects | Basic | Rich particle |

## Summary

**Brawler AGA** demonstrates character sprite enhancement.

### Technical Specifications

- **Display:** 256 colours, Y-sorted
- **Characters:** 64+ colours each
- **Stages:** 3 with full-colour backgrounds
- **Effects:** Enhanced combat visuals
- **Players:** 1-2 cooperative
- **Target:** A1200, A4000
- **Distribution:** Separate AGA ADF

### Skills Learned

- Large sprite colour enhancement
- Background/foreground palette balance
- Combat effect enhancement
- Multi-character AGA performance
